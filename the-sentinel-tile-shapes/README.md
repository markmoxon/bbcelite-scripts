# The Sentinel tile shape image generator

This folder contains a web page to generate images of tile shapes for use in The Sentinel's deep dives.

To use it, deploy both files to a web server, and then visit URLs like this to generate animated GIFs:

https://localhost/the-sentinel-tile-shapes/shapes.html?shape=10

Shape numbers are as in the documentation, i.e. in the range 0 to 15 (with the exception of 8), and with 4a, 4b, 12a and 12b supported.
